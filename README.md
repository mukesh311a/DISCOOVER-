# DISCOOVER: Spatiotemporal Trust Evaluation

This repository contains the code and the corresponding research paper for **Spatiotemporal Trust Evaluation**. 

## Contents

- `Spatiotemporal Trust Evaluation.ipynb`: A Jupyter Notebook containing the implementation and evaluation of spatiotemporal trust models. It includes the data processing, model evaluation, and results.
- `PAPER.pdf`: The corresponding research paper detailing the methodology, algorithms, and theoretical foundation of the spatiotemporal trust evaluation approach.

## Usage

To run the evaluation code:
1. Ensure you have Jupyter Notebook or JupyterLab installed.
2. Install the required Python dependencies (such as `numpy`, `pandas`, etc., as used in the notebook).
3. Open `Spatiotemporal Trust Evaluation.ipynb` and execute the cells.
